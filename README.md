# Lexi-AI

An intelligent document summarization web app powered by AI (OpenAI & Google Gemini).  
Upload PDFs or DOCX files and get concise, customizable summaries instantly.

## Features
- Upload PDF or DOCX files
- Choose AI model (OpenAI or Gemini)
- Control summary length (short, medium, long)
- Copy or download summarized results

## Tech Stack
- HTML5, CSS3, JavaScript
- PDF.js, Mammoth.js
- OpenAI & Gemini API integration
